Installation
-------------------------

- Clone the repo in from `rscvp <https://github.com/ytsimon2004/rscvp>`_

- Create conda env

.. prompt:: bash $

    conda create -n rscvp python=3.10 -y


- Create conda env

.. prompt:: bash $

    conda activate rscvp


- Installation of package

.. code-block:: bash

    pip install .[all]
