Local Installation
-------------------------

.. note::

    No need if you prefer run on the google colab notebook


1. Clone the repo from `rscvp <https://github.com/ytsimon2004/rscvp>`_

2. Create conda env

.. code-block:: bash

    $ conda create -n rscvp python=3.10 -y


3. Create conda env

.. code-block:: bash

    $ conda activate rscvp


4. Installation of package

.. code-block:: bash

    $ pip install .[all]
