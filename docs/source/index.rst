Installation
-------------------------

- Clone the repo from `rscvp <https://github.com/ytsimon2004/rscvp>`_

- Create conda env

.. code-block:: bash

    $ conda create -n rscvp python=3.10 -y


- Create conda env

.. code-block:: bash

    $ conda activate rscvp


- Installation of package

.. code-block:: bash

    $ pip install .[all]
